## Deploy to Railway

### Step 1 — Push to GitHub
```bash
git init
git add .
git commit -m "ResumeForge by Yashwanth Keerthi"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/resumeforge.git
git push -u origin main
```

### Step 2 — Deploy on Railway
1. Go to [railway.app](https://railway.app)
2. Click **New Project** → **Deploy from GitHub repo**
3. Select your repository
4. Railway auto-detects `package.json` and runs `npm start`
5. Your site goes live with a public URL like `resumeforge.up.railway.app`

### Step 3 — Custom Domain (Optional)
In Railway dashboard → Settings → Domains → Add custom domain
